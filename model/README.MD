# Selffy - AI Model API

## Tech Stack
- Node.js
- Express.js
- TypeScript
- Zod
- Ollama

## How to run it?
1. Go to the **model** folder:
```bash
cd model
```

2. Install the dependencies:
```bash
npm install
```

3. Install Ollama:
[https://github.com/ollama/ollama](https://github.com/ollama/ollama)

4. Install Tinyllama:
```bash
ollama pull tinyllama
```

5. Run the application:
```bash
npm run start
```
